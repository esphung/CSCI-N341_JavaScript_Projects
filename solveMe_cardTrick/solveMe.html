Task: Card Magic Trick

1.Show user 9x playing cards face up and say "Pick your favorite card without telling me what it is. When you are ready click Yes and I will find which card you are thinking of".

*Prompt user to click Yes or No ("Are you ready?")

*If Yes proceed to step 1a.

1a.Shuffle 9x standard playing cards.

*Arrange playing cards face down in 3 x 3 grid on Table. Label horizontally as three different rows (Row A, Row B, and Row C(respectively from top to bottom). Hide labels from user.

1b.Show the user Row A cards face up and prompt Yes or No ("Do you see your card?)

* If Yes proceed to step 10a.

* If No proceed to step 20a.

10a. Shuffle cards that did not contain the users card into new Row A, Row B, and Row C and be sure to begin each new row with the individual cards from the row in which the user selected Yes and was led to this step. The new rows are now labeled Rows D, E, and F. The other 6x cards in these rows should be completely random.

10b. Display Row D face up to user and prompt Yes or No ("Do you see your card?")

* If Yes. Show user first card in current Row and say "This is your card!"

* If No, repeat step 10b. with Row E, then Row F until user selects Yes at the prompt and then display the first card in current row and say "This is your card!"

20a. Shuffle Row A cards and set aside. Display Row B face up for user to see and prompt Yes or No ("Do you see your card?")

* If Yes proceed to step 10a.

* If No proceed to step 30a.

30a. Discard Row B into the pile with Row A and immediately move on to step 10a.
